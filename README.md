# BioFAIR GitHub Organisations Governance
Welcome to the BioFAIR GitHub Organisation Governance. Here we describe the set up of the GitHub organsiation with the rights and responsibilities of the members. Read on if you want more information on how to contribute to this repository.

BioFAIR is a Hub and Spoke model with several Commons: Data, People, Methods and Knowledge Hub. Members of the Commons will be actively contributing to the GitHub organisation.

## Base Member Permissions
If you are a member of the BioFAIR organisation you have the following permissions:
* Read public repositories only
* Can be invited to teams and private repositories in the organisation

## Teams 

The organization has separate teams for each of the Commons. When you are added to the organisation you will be invited to the appropriate team e.g. Methods Commons. Once you are added to the team you will have the following permissions:
* Read public repositories in the BioFAIR organisation
* Read private repositiories that belong to that team
* Create public and private repositories

Each team will have Maintainer privileges given to that Spoke's Project Lead/s. The Project Lead will then be able to assign permissions to the members of that team. 

## Owners
The owners of the organisation and the repositor will be part of the BioFAIR Hub team. For any changes at an organisational level please contact one of the owners of the repository.
